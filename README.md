### Compiles and hot-reloads for development
```
npm run dev
```

# Usage

## Development

Just run and visit [http://localhost:8080](http://localhost:8080)

```bash
npm run dev
```

## Build 

To build the App, run

```bash
npm run build
```

And you will see the generated files in `dist`, which are ready to be served.
